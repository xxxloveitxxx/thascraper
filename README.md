# 🌐 ThaScraper
**The ultimate bridge between your browser and AI intelligence.**

ThaScraper is a high-performance Chrome Extension that transforms the web into a structured data source for AI agents. By leveraging the Model Context Protocol (MCP), it enables seamless data extraction and real-time communication between your active browser session and your LLM's brain.

### ✨ Key Features
- 🚀 **Instant Web Extraction:** Effortlessly scrape page content with a single click.
- 🧠 **MCP Integration:** Built-in support for the Model Context Protocol, allowing AI agents to "see" and "interact" with your browser.
- ☁️ **Cloud-Powered Backend:** Seamlessly connects to a dedicated MCP server for robust data processing.
- 🛠️ **Agent-Ready:** Designed specifically for developers building autonomous AI workflows.

### 🛠 How It Works
ThaScraper acts as the "eyes" for your AI. It captures the DOM and essential metadata from your current tab and forwards it to the **ThaScraper MCP Server**, which then translates that raw data into a format that LLMs can use to perform complex tasks.

### 🚀 Installation
1. Clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the `newscraper` folder.

### 💻 Tech Stack
- **Frontend:** JavaScript (Chrome Extension API)
- **Protocol:** Model Context Protocol (MCP)
- **Backend:** Hosted on Render
